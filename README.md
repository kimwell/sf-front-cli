## 安装

```shell
npm install sj-front-cli -g
```

## 使用

`sj-front-cli`命令符为`sj-front`

```sh
# 初始化项目
sj-front init

# 填写 项目名称、项目描述
? 请输入项目名称： [项目名称]
? 请输入项目描述： Base on sj-front-cli

# 进入项目
cd [项目名称]

# 安装依赖
npm install

# 运行
npm run dev

# 启动访问
http://localhost:3000/
```
